## Install and run:
- `cp .env.example .env`
- `make up`
### Set up your database  
#### only if you don't want use make up
- `php artisan migrate`
- `php artisan db:seed`
- `php artisan serve`
- `php artisan queue:work`
- `npm install'
- `npm run dev`
- Login with `demo@mailtool.test` and `password`
